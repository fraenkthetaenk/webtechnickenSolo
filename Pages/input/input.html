Für die Erstellung der Seite wurde das Flexbox modell verwendet.
Das Flexbox modell ist der aktuelle Standard für das Erstellen von Layouts.
Flexbox kann dazu verwendet werden um Layouts nach Spalten oder nach Reihen erstellt werden.
Die Flexbox wird dadurch bestimmt, dass im außeren Element die Display Eigenschaft auf flex gesetzt wird.
Ob man nun ein Layout nach Spalten oder nach Reihen erstellt, wird durch die Eigenschaft flex-direction festgelegt. Hier kann man auch noch wählen ob die Elemente in Reihenfolge oder umgekehrt dargestellt werden soll. Die Reihenfolge der Elemente entspricht standarmäßig der in der sie im Sourcedokument vorkommen, allerdings, kann man diese sspäter auch noch mit der Eigneschaft order beliebig ändern. Wenn mehrere Elemente denselben order wert haben entschiedet wirder die Position im Dokument.
Im Gegensatz zum Float Layout haben alle Elemente innerhalb einer Flex-Elementes dieselbe Höhe.
Über die Eigenschaften Flex-grow und flex-shrink der Kinder kann man zudem bestimmen wie schnell sie im Verhältnis zu Ihren Geschwistern wachsen und/oder schrumpfen.
Die eingeschaft flex-basis kann dazu genutzt werden um einen minimale Größe für das Element (in der Aktuellen Flex-richtung) zu definiern, den sie auch nicht unterschreiten dürfen. Bevor sie diesen unterschreinten, werden erst die Geschwister geschrumpft, auch wenn diese eigentlich langsamer schrumpfen würden.
