Das Layout dieser Seite ist ein Grid-Layout
Das Grid Layout ermöglicht es Elemente gleichzeitig nach Spalten und nach Reihen anzuordnen.
Diese Art der Erstellung befindet sich noch in der Entwicklung und ist wird aktuell nur von 75% der genutzten browser unterstützt. Stand 30.11.17 nach caniuse.com
Wrapper:
display: grid;
grid-template-columns:
grid-auto-rows:
Die Elemente positionieren sich dann entweder nach Sourc-Reihenfolge oder können mit den css-properties: grid-column und grid-row einer Position zugewiesen werden. Hierbei kann man diese Elemente auch über mehrere Zellen laufen lassen.
Alternativ hierzu kann man dies auch über die Verwendung von der Eigenschaft Grid-area machen. Mit dieser kann man ein Grid durch die verwendung von Key-words erstellen und den Elementen dann die Keywordszuweisen, deren Darstellung sie annhemen sollen.

Mithilfe von Grid Areas, kann man Rechteckige bereiche des Grids definieren und diese den Grid-Elementen zuweisen.
Dadurch muss man sich nicht mehr überlegen über welche Reihen und Spalten sich das Element erstreckt, sondern einfach nur definerne welchen bereich sie beschreiben.
Für das nebenstehende Layout wurde folgens Area Definiert
"1 1 2"
"1 1 3"
"4 4 3"
